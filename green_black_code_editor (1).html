مثال‌های قابل شناسایی:

hacking oktay parsa → نمایش: «اوکتای پارسا هک شد»
hacking نام → نمایش: «<نام> هک شد»
say <متن> → نمایش متن ساده

با زدن دکمه «اجرا» هر خط پردازش می‌شود. برای هر خط که الگوی «hacking» پیدا شود، یک صدای کوتاه پخش می‌شود (شبیه افکت).
